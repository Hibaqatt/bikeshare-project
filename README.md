\# Bikeshare Project



\## Description

This project analyzes bikeshare data for different US cities using Python.



\## Files

\- bikeshare.py: main program for data analysis

\- chicago.csv: dataset for Chicago

\- new\_york\_city.csv: dataset for New York City

\- washington.csv: dataset for Washington



\## Features

\- Filter data by city, month, and day

\- Display statistics on travel times

\- Analyze user types and stations



\## How to run

Run the bikeshare.py file using Python.
