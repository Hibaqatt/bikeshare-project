Project: Explore US Bikeshare Data

This project analyzes US bikeshare data for Chicago, New York City, and Washington using Python.

The analysis was implemented using:
- Python
- pandas
- NumPy

Data processing and statistical analysis were performed using the following documentation resources:
- Udacity course materials (Pandas & NumPy lessons)
- pandas documentation: https://pandas.pydata.org/docs/
- NumPy documentation: https://numpy.org/doc/

The script allows interactive filtering by city, month, and day, and computes descriptive statistics such as:
- Most frequent travel times
- Most popular stations and trips
- Trip duration statistics
- User statistics
